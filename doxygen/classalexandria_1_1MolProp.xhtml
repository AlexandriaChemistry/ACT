◆ validate()

void alexandria::MolProp::validate ( MsgHandler * msghandler)

Check the internal consistency of this object.

Reports the number of experiments via msghandler at Info level. Reports errors if experiments with atoms have inconsistent atom order (name, obtype or atomid differ between experiments). For dimers (molecules with exactly two fragments) reports errors if any bond connects atoms belonging to different fragments.

Parameters
[in]msghandlerHandler for logging and error reporting
