alexandria::ForceComputer Class Reference

#include <act/forces/forcecomputer.h>

Description

Class to compute all the forces in a molecule or complex. It is imperative that there is only one force computer in each program, since results may depend on the flags given when initiating. This goes in particule for the maxShellDistance.

Do complete energy/force computation. If shells are present their positions will be minimized. If the shell minimization does not converge a warning will be written.

Parameters
[in]msg_handlerFor debugging, may be nullptr
[in]pdPointer to force field structure
[in]topThe molecular topology
[in]coordinatesThe atomic coordinates. Coordinates of shell particles may be changed.
[out]forcesThe atomic forces
[out]energiesThe energy components
[in]fieldOptional electric field to be applied
[in]resetShellsSet the position of the shells to that of the connecting atoms
[in]relaxSpecify the shell indices that should be relaxed. If empty, all shells will be relaxed.
